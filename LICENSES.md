# Repository Licensing Notes

This repository is a composite research prototype, so license boundaries matter:

- The root `LICENSE` is the BSD-3-Clause license carried over from the IDC tutorial materials that seeded this workspace.
- New GrandQC-IDC wrapper code in `modules/`, `web/`, `scripts/`, and `notebooks/` is intended to be used under BSD-3-Clause terms unless a file says otherwise.
- The cloned `grandqc/` reference implementation remains under its own license in `grandqc/LICENSE` and is treated as read-only reference code.
- GrandQC pretrained checkpoints are not committed here. Their upstream license and usage restrictions, including non-commercial restrictions where applicable, control any downloaded weights.
- IDC image data are not redistributed by this repository. Data accessed through IDC keep their collection-level licenses as reported by IDC.

For publication, deployment, or commercial use, review the upstream GrandQC model/license terms and the IDC collection license for each cohort.
